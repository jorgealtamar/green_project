# Green Project

## Descripción

**Green Project** es una aplicación móvil creada en Flutter para identificar tipos de residuos y guiar a los usuarios sobre dónde depositarlos correctamente dentro del campus universitario. La app utiliza un modelo de reconocimiento de imágenes creado con Teachable Machine para facilitar esta identificación.

## Estado del Proyecto

Este proyecto está actualmente en construcción.

## Características

- **Identificación de Residuos:** Usa la cámara de tu dispositivo para identificar diferentes tipos de residuos.
- **Guía de Depósito:** Informa al usuario sobre la ubicación correcta de los contenedores dentro del campus universitario.
- **Modelo de IA:** Implementado con Teachable Machine para el reconocimiento de residuos.

## Instalación

Para clonar y ejecutar esta aplicación, necesitas tener instalado Flutter en tu máquina. Sigue los siguientes pasos:

1. Clona este repositorio:
   ```sh
   git clone https://github.com/jorgealtamar/green_project.git

2. Navega al directorio del proyecto:
   ```sh
   cd green_project

3. Instala las dependencias:
   ```sh
   flutter pub get

4. Conecta tu dispositivo o inicia un emulador:

5. Corre la aplicación
   ```sh
   flutter run

## Uso

1. Abre la aplicación.

2. Usa la cámara para tomarle foto a un residuo.

3. La aplicación identificará el tipo de residuo y te indicará dónde debes depositarlo dentro del campus.
